# ERDDAP Proxy API

Simple Node-Express API to fetch and process data from ERDDAP.

To install:
`yarn`

To run:
`yarn start`
`yarn dev` (if trying to connect to a local erddap instance, use `yarn dev:local` instead)

Linting:
`yarn prettify`

Listening on:
`localhost:8088`

Documentation:
`localhost:8088/api-docs`

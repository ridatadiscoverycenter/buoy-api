# ERDDAP Proxy API

Simple Node-Express API to fetch and process data from ERDDAP.

To install:
`npm install`

To run:
`npm run start`
`npm run dev` (if trying to connect to a local erddap instance, use `npm run dev:local` instead)

Linting:
`npm run prettify`

Listening on:
`localhost:8088`

Documentation:
`localhost:8088/api-docs`
